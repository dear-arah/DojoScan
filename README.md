## ğŸš€ Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- React Native development environment set up
- iOS Simulator / Android Emulator or physical device

### Installation

1. Clone the repository:
```bash
git clone https://github.com/dear-arah/DojoScan.git
cd DojoScan
```

2. Navigate to the frontend directory:
```bash
cd frontend
```

3. Install dependencies:
```bash
npm install
# or
yarn install
```

4. Run the application:

For iOS:
```bash
npm run ios
# or
yarn ios
```

For Android:
```bash
npm run android
# or
yarn android
```
